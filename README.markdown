# NumToPersian

Convert numbers to Persian letters in  JavaScript

#تبدیل عدد به حروف فارسی در جاوا اسکریپت

#No need to jQuery!

#Tested and reliable

#Easy to use

just Call `NumToPersian` or use `toPersian()` Prototype.

#Example

```javascript
<script>
  alert(NumToPersian(1250));
  "1250".toPersian();
</script>
```
